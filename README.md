# Reporting

In the project directory, you can run:

### `python manage.py runserver`
Runs the app in the development mode.  
Open [http://127.0.0.1:8000/reports/generate/] to view it in your browser.  

The server will reload when you make changes.  
You may also see any errors in the console.

---

### `python manage.py migrate`
Applies database migrations.

---

### `python manage.py makemigrations`
Creates new database migrations based on the changes you have made to your models.

---

### `python manage.py createsuperuser`
Creates an admin user to access the Django admin panel.

---

### `python manage.py test`
Launches the test runner for the project.
